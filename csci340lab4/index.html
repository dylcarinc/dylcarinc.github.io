About This Site

This website calls the last.fm API to get the 50 most recent tracks that I have listened to on my lastfm account. It then calls the wikipedia API to get information about the artist of the song.

There are still some issues with the Wikipedia API, as it will return a page about the bird, rather than the band the Eagles. I hope to add the ability to put in other usernames too.
